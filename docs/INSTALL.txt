mingtak.loginmethod Installation
--------------------------------

To install mingtak.loginmethod using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``mingtak.loginmethod`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        mingtak.loginmethod
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
